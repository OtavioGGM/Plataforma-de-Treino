A ideia do projeto é ser um HTML simples (futuramente PWA) com o objetivo de auxiliar no treino de musculação.
Ao final do projeto a plataforma deverá fornecer lista de exercícios editável, temporizadores, tabelas sobre suplementos e entre outras funcionalidades úteis para pessoas com interesse em otimizar seu treino e dia a dia.

URL do projeto: https://otavioggm.github.io/Plataforma-de-Treino/
