PLATAFORMA DE TREINOS PARA ACADEMIA

*Aplicação web simples, desenvolvida inicialmente em HTML, com evolução planejada para Progressive Web App (PWA), voltada ao acompanhamento de treinos e alimentação focados em hipertrofia e ganho de massa muscular. O objetivo do projeto é oferecer ferramentas práticas para auxiliar na organização e otimização da rotina de musculação.

FUNCIONALIDADES PLANEJADAS:

*Lista de exercícios editável
*Temporizadores para descanso e execução
*Tabelas informativas sobre suplementos
*Organização básica de treinos
*Interface simples e acessível
*Suporte futuro para funcionamento offline (PWA)


OBJETIVO DO PROJETO:

*Este projeto tem caráter educacional e prático, servindo como base para o desenvolvimento de aplicações web progressivas, explorando conceitos de interface, usabilidade e funcionalidades úteis para praticantes de musculação.

TECNOLOGIAS UTILIZADAS:

*HTML
*CSS
*JavaScript
*PWA (implementação futura)
